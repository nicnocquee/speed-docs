title: Steps
---

You can render steps in your documentation by using the `Steps` component.

```mdx title="steps.mdx"
---
title: Steps
---

import { Step, Steps } from "fumadocs-ui/components/steps";

<Steps>
<Step>

### This is step 1

</Step>

<Step>

### This is step 2

</Step>
</Steps>
```

which will be rendered as:

import { Step, Steps } from "fumadocs-ui/components/steps";
