---
title: How it works
---

- Prepare your documentation content in [MDX format](https://mdxjs.com/). See [Preparing your content](/preparing-your-content) for more details.
- Run the `speed-docs` CLI tool to build your **static** documentation. Make sure you have installed the `speed-docs` CLI tool. See [Installation](/#installation) for more details.
- Deploy your documentation to any static hosting service. See [Deploying your documentation](/deploying) for more details.
